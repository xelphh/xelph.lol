# Getting Started with Python

Python is one of the most beginner-friendly programming languages. Here's how to get started.

## Installation

Download Python from [python.org](https://www.python.org/downloads/) and follow the installer.

## Your First Program

```python
print("Hello, World!")
```

## Variables and Types

```python
name = "Xelph"
age = 25
height = 5.9
is_developer = True
```

## Control Flow

```python
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")
```

Happy coding! 🚀
